# About the project

This project examines the circulation of **xuexiji** (学习机, “learning machines”) in mainland China. From the 1980s to the early 2000s, a wide range of these devices proliferated across the country. Drawing on technologies associated with Taiwan’s Microprofessor company and assembly practices in the Pearl River Delta, these machines were distributed nationwide through several key commercial and logistical hubs. This is an ongoing project.

# About the author

Weilin Zhu (https://mediastudies.as.virginia.edu/people/weilin-zhu) is a Ph.D. student in Media Studies at the University of Virginia. Her research examines the history of computing, with particular attention to the development of computing technologies in post-socialist China, including microcomputers, educational learning machines, and bootleg game consoles in the emerging information era. More broadly, her research engages media infrastructure, science and technology studies (STS), digital humanities, artificial intelligence, and environmental humanities. She is currently researching early Internet and pre–World Wide Web network practices in 1990s China, focusing on alternative, non-PC computing platforms as localized sites of Internet access and use.

# About the Media Mapper

## What is it?

The Media Mapper is an open source framework intended to be used as a starting point for building a web application geared towards exploring Media throughout time and space.

The project was dreamt up by **Ennuri Jo** as a way to explore the relationship between media and water. Funding for this project was provided by the [University of Pennsylvania, Penn](https://www.upenn.edu/). The framework was developed by Ben Tyler and [Lost Creek Designs](https://lostcreekdesigns.co/).

Please see the [GitHub repository](https://github.com/Aqueous-Earth-Catalog-Team/media-mapper) for more information about how you can get started with the project!
